同源策略
        1. 同源策略 Same-Origin-Policy 最早是由 Netscape 公司提出的一种安全策略
        2. 同源: （1）协议相同
                 （2）域名相同
                 （3）端口号相同
                 * 必须完全相同 *
        3. 违背任何一条就是同源策略就是 “跨域”
        note: * ajax 默认遵守同源策略 *


        eg. 
        http a.com 8000 port 《====》  http a.com 8000 port  完全一致

        同源策略 根本 -> 同一个来源
